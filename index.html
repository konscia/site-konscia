Konscia // Software

Olá, sou Kaléu Puskas. Profissionalmente identifico minha empresa como Konscia, uma palavra em esperanto que significa consciente. Estar consciente é uma busca constante, comigo e com a tecnologia.

A Konscia desenvolve soluções de software para a internet como portais, softwares de gestão e aplicações online.

O objetivo é transformar conhecimento em tecnologias escaláveis.

Isso envolve organizar bases de dados, construir visualizações humanizadas, sistematizar metodologias, compreender gestão empresarial e desenvolver conexão com outras tecnologias via API’s.
